Overview of UF Digital Library General Tools Class Library

What follows below is a breakdown of all the major classes which are directly used when interfacing with the classes in this library. A short explanation of the use of the class as well as links to associated documentation are also provided.



Major Classes in the UF Digital Library General Tools Class Library

Class Namespace Description
CD_Packer UF.Libraries.DLC.GeneralTools The CD_Packer object is used to pack CD's from a provided directory, or adds each file individually.

VersionChecker UF.Libraries.DLC.GeneralTools.StartUp VersionChecker is a class used to check against a XML versioning database file on the network drives to determine if a newer version of an application is available.

RemarkedDataSet UF.Libraries.DLC.GeneralTools RemarkedDataSet extends the DataSet object and includes remarks for each data element.

OleDatabaseWrapper UF.Libraries.DLC.GeneralTools OleDatabaseWrapper is the class which is used for manipulation of an Access database.

TiffInfo UF.Libraries.DLC.GeneralTools TiffInfo is an object which reads an existing TIFF image and can then be queried for the header information.

LogFileText UF.Libraries.DLC.GeneralTools.Logs A log file IO control object which writes to a text-based log file.

LogFileXHTML UF.Libraries.DLC.GeneralTools.Logs This class establishes a rich XHTML log file. Then, lines can be added in three flavors ( error, nonerror, and complete).

DLC_SecurityInfo UF.Libraries.DLC.GeneralTools.StartUp [DO NOT USE - UNDER CONSTRUCTION]

DLC_SecurityInfo checks for DLC specific domain user groups in the SMATHERSLIB domain.

SecurityInfo UF.Libraries.DLC.GeneralTools.StartUp Object used to determine and ensure security. It allows for reading from the registry, checking local users and computer information, and writing and reading to encrypted files.

SingleProgramInstance UF.Libraries.DLC.GeneralTools.StartUp SingleProgramInstance is an object which is used to ensure that only one instance of this application is running at one time.

DirectoryCrawler UF.Libraries.DLC.GeneralTools.IO DirectoryCrawler is a class which will check a directory, and every subdirectory, and compile a list of all the files.

Connected_Drives UF.Libraries.DLC.GeneralTools.IO Class allows access to all the drives connected to the local computer.

Volume_Folders UF.Libraries.DLC.GeneralTools.IO Class is used to search for valid volume folders.
